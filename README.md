# 헌책 헌터

## 알라딘 중고서점 검색기 - 헌책 헌터

원하시는 중고책이 어디에 있는지 빠르게 찾아볼 수 있도록 마련된 도우미입니다  
한번의 검색으로 서울 지역에 있는 모든 알라딘 중고서점의 책을 검색해드립니다  
(다만 시간이 조금 걸리는 점 양해바랍니다)

* Ruby version: 2.3.1

* Rails version: 5.0.0.1

* Configuration

    bundle install

* Database initialization

    rails db:create # 사실 DB는 전혀 쓰지 않는 어플리케이션입니다
    rails db:migrate
